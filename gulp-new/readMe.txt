Запуск нового проекта:
1. Переименовать папку
2. Запустить VSCode
3. Добаваить папку в VSCode
4. Открыть терминал
5. npm i
6. Переустановить: npm i webp-converter@2.2.3 --save-dev
7. Переустановить: npm i gulp-ttf2woff2 --save-dev
------------------------------------------------------

Шрифты:
- для конвертации шрифтов из otf в ttf запустить: gulp otf2ttf
- для стилизованных файлов изменить в файле 'fonts.scss':
	- название на одинаковое
	- font-weight на нужный параметр
------------------------------------------------------

SVG-sprite (если нужен SVG-sprite)
- в паку app/iconsprite/ добавить нужные иконки
- запустить в отдельном терминале: gulp svgsprite
- нужный файл будет скомпилирован в папку 'dist'/img/icon/
- пример файла будет скомпилирован в папку 'dist'/img/stack/
